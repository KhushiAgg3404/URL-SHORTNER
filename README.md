# URL Shortener

A full-stack URL Shortener application built with the MERN stack that allows users to generate short URLs, create custom slugs, track clicks, and manage all their links through a personalized dashboard.

## Features

* Generate short URLs from long links
* Support for custom URL slugs
* Secure JWT-based authentication
* URL analytics with click tracking
* Copy shortened URLs with a single click
* Responsive and user-friendly interface
* Protected user dashboard
* Real-time data synchronization using React Query
* Full-stack MERN architecture

## Backend Architecture

The backend follows a layered architecture pattern:

* **Routes** в†’ Define API endpoints and handle request routing.
* **Controllers** в†’ Process incoming requests and manage responses.
* **Services** в†’ Implement business logic and application workflows.
* **DAO (Data Access Objects)** в†’ Interact with the database and perform data operations.
* **Models** в†’ Define MongoDB schemas and data structures.
* **Middleware** в†’ Handle authentication, authorization, and request processing.
* **Utils** в†’ Provide reusable helper functions and utilities.

## Authentication Flow

1. User registers or logs in.
2. JWT token is generated by the server.
3. Token is stored securely in cookies.
4. Protected routes verify authentication.
5. Users can create and manage custom URLs.
